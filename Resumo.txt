Type system
  Sistema de tipos que utilizamos para descrever nossos dados

Queries
  Obtém dados da API (read)
    GET

Mutations
  Faz alterações nos dados da nossa API (write)
    POST - PUT - DELETE

Schema
  Define o esquema da API, pense nele como um container para todos os tipos da API (SDL)
  Engloba as Queries, Mutations, Subscriptions, Directives e etc

Resolver
  Cada campo no GraphQL possui uma função resolver
  Todo campo no GraphQL é resolvido após o retorno de um valor escalar (Scalar Types)
